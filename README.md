# Holiday Cheer RStudio Theme

by: Elisa D'Amico

A festive dark theme for RStudio featuring warm holiday colors with reds, greens, and gold accents. Perfect for coding through the holiday season!

## Features

- **Fully dark interface** 
- **Christmas color palette**:
  - Deep warm brown/black backgrounds
  - Crimson red keywords and operators
  - Forest and lime greens for strings and comments
  - Gold and golden yellow for functions and important elements
- **Festive tabs** - Dark forest green inactive tabs with gold close buttons
- **Beautiful syntax highlighting** - Carefully chosen colors for optimal readability
- **Comfortable selection highlighting** - Visible brown-red selection with distinct word highlighting


## Installation

1. Download the `Christmas_Cheer.rstheme` file from this repository
2. Open RStudio
3. Go to **Tools ‚Üí Global Options ‚Üí Appearance**
4. Click **Add...** under the Editor theme section
5. Navigate to and select the downloaded `Christmas_Cheer.rstheme` file
6. Click **Apply** to activate the theme

## What's Themed

- Code editor with festive syntax highlighting
- Console with green input text
- All UI panels (Environment, Files, Plots, etc.)
- Tabs and tab close buttons
- Dialogs and popup windows
- Buttons and input fields
- Scrollbars
- Tables and data viewers
- Toolbars and menus

## Customization

Feel free to fork this repository and customize the colors to your liking! The theme file is a standard CSS file that you can edit with any text editor.

### Key Color Values

- Background: `#1a1512` (warm dark brown)
- Keywords: `#dc143c` (crimson red)
- Strings: `#90ee90` (light green)
- Functions: `#ffd700` (gold)
- Comments: `#8fbc8f` (sea green)
- Selection: `#a52a2a` (brown-red)
